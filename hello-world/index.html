反射:

反射库是java中能够动态的操纵java代码的程序，这种能分析类能力的程序被称为反射(reflective).
那么反射具体有哪些反射机制呢？

首先来说Class类

Class 类

在程序运行期间，有着记录对象所属的类的信息的类，那就是Class类。

总结

java.lang.Class
1.static Class forName(String className)
返回描述类名为className的Class对象。
2.Object newInstance()
返回这个类的一个新实例。
3.Class getClass()
返回这个类的Class对象。
4.String getName()
返回该Class对象的类名。
